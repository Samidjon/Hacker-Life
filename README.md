# Hacker Life

Hacker Life is a browser-based cybersecurity career simulator built with React and TypeScript. Start as a beginner, complete interactive contracts, earn credits and experience, unlock skills, purchase upgrades and grow your reputation.

## Features

- Interactive cybersecurity contracts
- Level and career progression
- Skill tree with permanent bonuses
- Hardware and software upgrades
- Energy regeneration and offline progress
- Achievements and rewards
- Daily missions and login streaks
- Automatic progress saving with `localStorage`
- Responsive hacker-style interface

## Technologies

- React
- TypeScript
- Vite
- CSS
- LocalStorage
